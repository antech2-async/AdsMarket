## What This Pack Is For

Pack ini dibuat agar NotebookLM menghasilkan pitch deck yang:

- patuh guideline OpenClaw Agenthon 2026, terutama maksimal 5 slide;
- menonjolkan AI Agent dan Multi-Agent System, bukan sekadar dashboard;
- kuat untuk additional track Best Payment Use Case;
- mudah dipahami juri non-technical dalam waktu singkat;
- cukup kredibel untuk juri technical;
- punya visual direction yang premium, fintech, agentic, dan tidak generic.

## Core Positioning

AdSourcing bukan marketplace iklan.

AdSourcing adalah autonomous sponsorship agent pair:

- SponsorAgent diberi mandat budget, policy, dan target komunitas.
- CommunityAgent diberi mandat inventory, floor price, dan content rules.
- Keduanya menjalankan deal end-to-end: handshake, verify, negotiate, escrow, delivery, verification, settlement, dan proof.

## Important Guardrails

- Jangan menyebut produk sebagai "marketplace iklan biasa".
- Jangan membuat klaim production-ready penuh.
- Jangan mengklaim DOKU live payment sudah production apabila hanya sandbox.
- Jangan mengklaim agent bebas tanpa batas. Agent bekerja di bawah mandate dan deterministic policy rails.
- Jangan membuat deck terlalu teknis seperti code walkthrough.
- Jangan memakai visual generic AI seperti robot, glowing brain, neon network, crypto coin spam, atau dashboard gelap penuh angka.
- Jangan memakai desain token dari SpecHeal.
- Jangan membuat deck campur bahasa secara acak. Bahasa utama harus Indonesia; istilah teknis English hanya untuk noun yang memang umum dipakai.
- Jangan membuat visual terlalu seperti dokumen hukum, arsip tua, atau stempel ramai. Sponsorship harus terasa sebagai transaksi komunitas yang hidup, bukan paperwork.
